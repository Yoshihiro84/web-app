Save to Papers Manager

Zotero Connectorのように、論文ページからワンクリックで登録できます。
Safari / Firefox / Chrome すべてで動作します。

方法1: ドラッグ&ドロップ

このボタンをブックマークバーにドラッグ:

Save to Papers

方法2: 手動登録（Safari向け）

  1. このページをブックマークに追加（Cmd+D）
  2. ブックマークを編集 → URLを下のコードに置き換え

使い方

  1. 論文のページを開く（arXiv, Google Scholar, 出版社サイトなど）
  2. ブックマークの「Save to Papers」をクリック
  3. 右下にパネルが表示される → メタデータを確認して「Save Paper」

対応サイト

  • arXiv — タイトル、著者、abstract、arXiv ID
  • Google Scholar — 検索結果の最初のエントリ
  • 出版社サイト — HTMLメタタグ（citation_title, DC.title 等）から自動抽出
